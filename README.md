# kintone カスタマイズ管理

このリポジトリは、kintoneアプリのJavaScriptカスタマイズコードを管理するためのものです。

## 概要

kintoneに追加するJavaScriptコードをGitHubでバージョン管理します。

## ファイル

### red.js

kintoneアプリに追加するJavaScriptカスタマイズコード。

## 使い方

1. コードを修正する
2. GitHubにコミットする
3. kintone管理画面からJavaScriptファイルをアップロードする

## 管理ルール

* コードを変更したらGitHubに保存する
* 変更履歴はcommitで管理する
* 本番環境に反映する前にテストする
